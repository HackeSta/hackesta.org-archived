Background

When I first learnt using Visual Studio, Engineer's Calculator was my first proper app. I used to play on West Side RPG Server back then, and I made this for that server.
The server had an Engineer job, which buys minerals from a MineWorker, and converts it to Guns, Ammo and some other stuff and sells it. This app was made for calculating the Mineral Requirements and Cost Price of manking something.
A similar app existed on the server's forums, but it wasn't updated for a while and missing a lot of things.
It took me around 4-5 hours to make the first version. I wasn't a good coder and all the code was uncommented, and I didn't knew about Arrays and some other stuff at that time, which resulted in a pretty long code, and made it very difficult for me to keep updating it. Although, I have added many new features since its release, but never did any huge changes.


Tutorial

Using the app is pretty simple, just choose the item you want, add the required values, press calculate amount, and them press calculate price.
