Background

When Pokemon GO came out, many developers created variety of tools. Many CP Calculators were available on the Internet, but none was for Android. So I decided to make one. Although I had never made a proper app for Android. I had learnt SQL recently, and made simple practice apps, which helped me a lot in this app.
Initially, I had used the graphics from the game, and not knowing about Vectors, PNG files were used. After that, a user on Reddit posted First Generation Pokemon Vectors by The Artificial and I switched to them.
The first version was only capable of one way calculation. Then someone recommended reverse calculation, which I named "Prevolution Mode". The database was at first made from compiling different data sources, but later I used Pokemon GO Hub's database.
In the recent update, I added multiple themes, whose icons were provided by Meritt Thomas


Features

  • Zero Adverts
  • Multiple Themes
  • Database by Pokemon GO Hub
  • Pokemon Images by The Artificial
  • Team Icons by Meritt Thomas
  • Two different estimation modes
  • Can Swipe to change Pokemon Evolutions
  • Help Mode can be accessed via the toolbar
